# Progect-RCTD
> 안드로이드 빌드 및 배포를 목적으로한 게임 프로젝트 입니다.

타워 소환, 조합, 강화를 통해 밀려오는 웨이브의 몬스터를 막으면서 마지막 웨이브를 막으면 클리어 할 수 있는 체스말과 랜덤성을 컨셉으로한 타워 디팬스 장르의 게임이며, 안드로이드 빌드 및 배포를 목적으로 하고 있습니다.
